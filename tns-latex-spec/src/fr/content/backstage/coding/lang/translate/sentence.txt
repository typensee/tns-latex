=================
Juste des phrases
=================

Les lignes suivantes extraites du fichier path::``FR.peuf`` de ¨bdoc montrent comment traduire des phrases "fixes" : tout se passe via un système de type peuf::``id_interne = La traduction`` (noter au passage la possibilité de faire des retours à la ligne dans une traduction tant que l'indentation de l'identifiant est respectée).

peuf::
    sentence::
        code_after_output = Le rendu précédent a été obtenu
    	                    via le code suivant.
    	output_after_code = Rendu du code précédent.


warning::
    Les identifiants comme peuf::``code_after_output`` et peuf::``output_after_code`` sont de la responsabilité du codeur et non du traducteur.
