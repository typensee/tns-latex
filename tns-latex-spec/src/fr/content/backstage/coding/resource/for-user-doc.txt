=============================
Des fichiers utiles à la ¨doc
=============================

Dans une ¨doc "utilisateur", il est très pratique d'user d'exemples stockés dans des fichiers externes : on peut penser à des images, de fichiers ¨tikz pour un graphique... ¨etc.
Le système retenu pour intégrer ces fichiers à la ¨doc finale s'appuie sur l'option term::``--recorder`` des compilateurs de la famille ¨tex : cette commande permet, entre autres choses, de connaître les fichiers externes utilisés pour compiler un document ¨latex.


warning::
    Bien respecter les règles de nommage des dossiers et fichiers vis à vis de ce qui est ignoré automatiquement : voir cf::``:?/auto-ignored.txt``.
