==============
\Refs internes
==============

Le nommage des \refs internes suit le schéma ci-dessous afin d'éviter tout conflit.

    1) Un premier préfixe est utilisé pour indiquer le nom du package comme dans ``tnsfictive-...`` par exemple.

    1) Si la \ref est relative à une macro spéciale comme ``\section``, ou un \env spécifique tel que ``\begin{doclatex} ... \end{doclatex}`, alors un \2e préfixe est utilisé comme dans ``tnsfictive-macro-section-...`` et ``tnsfictive-env-doclatex-...`` par exemple.

    1) Pour finir vient un \id laissé au libre choix du codeur.
