==============
\Refs internes
==============

Le nommage des \refs internes suit le schéma ci-dessous afin d'éviter tout conflit.

    1) Un premier préfixe est utilisé pour indiquer le nom du package comme dans ``tnsfictive-...`` par exemple.

    1) Si la \ref est relative à une macro spéciale comme ``\eval``, ou un \env spécifique tel que ``\begin{bdoclatex} ... \end{bdoclatex}`, alors un \2e préfixe est utilisé comme dans ``tnsfictive-macro-eval-...`` et ``tnsfictive-env-bdoclatex-...`` par exemple.

    1) Pour finir, un \id laissé au libre choix du codeur est ajouté.
