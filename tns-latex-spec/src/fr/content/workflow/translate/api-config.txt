=====================================
Langues pour les ¨macros et les ¨envs
=====================================

Prenons le cas du package ¨bdoc dont le code source contient le dossier de ¨dev path::``main`` suivant.

path-tree::
    + main
        + locale
            * FR.sty
            + fr
                * macros.txt
                * sentences.txt


De façon automatisée, un dossier path::``contribute/api/fr/main``, comme ci-dessous, contiendra la dernière version des traductions utiles au fonctionnement du package ¨bdoc.

path-tree::
    + tnsfictif
        + translate
            + api
                + fr
                    + main
                        * macros.txt
                        * sentences.txt
            + [...]
        + [...]


Il ne reste plus qu'à un traducteur allemand, par exemple, à créer un dossier path::``de`` avec le même contenu que celui du dossier path::``fr``.
La ¨trad sera alors simple comme le montre le contenu du fichier path::``sentences.txt`` donné ci-après.

verb::
    ---
    path = :?../bc-tools/for-latex/bdoc/src/main/locale/fr/sentences.txt
    ---


warning::
    Les identifiants comme verb::``code_after_output`` et verb::``output_after_code`` sont de la responsabilité du codeur, et non de celle du traducteur.
