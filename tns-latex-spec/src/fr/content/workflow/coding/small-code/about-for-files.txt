=============================================
path::``about.yaml`` pour les fichiers source
=============================================

Dans chaque dossier ¨src peut être mis un fichier path::``about.yaml`` pour indiquer les fichiers à analyser suivant un ordre imposé.
Dans cette section, nous allons nous concentrer sur les fichiers path::``STY`` liés au code, mais nous reviendrons sur les fichiers path::``TEX`` pour la ¨doc "utilisateur" dans la section suivante.
Imaginons que l'on ait le code suivant pour notre exemple ficitf.

yaml::
    ---
    title = Contenu du fichier path::``src/poor-lonesome-cowboy/about.yaml``
    ---

    toc:
        - rantanplan.sty
        - lucky.sty


Le code précédent sert à considérer le fichier path::``rantanplan.sty`` avant path::``lucky.sty``, et non suivant l'ordre alphabétique qui aurait été retenu sans emploi du fichier path::``about.yaml``.
