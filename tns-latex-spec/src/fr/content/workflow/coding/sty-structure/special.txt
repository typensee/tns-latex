==============================
Organiser des sections de code
==============================

Nous veons de voirs que les outils automatisés utilisent des commentaires de type "section à la ¨markdown".
Il est auss possible d'utiliser un autre type de sections pour faciliter la maintenance et la compréhension d'un code comme indiqué ci-dessous.

latex::
    % == TITRE SPÉCIFIQUE POUR L'AUTOMATISATION == %

    % ...

    % -- Titre laissé au libre choix du codeur -- %

    % ...


warning::
    Les titres pour les tâches automatisées sont fixés suivant les cas d'utilisation et donc ils ne peuvent pas être modifiés.
