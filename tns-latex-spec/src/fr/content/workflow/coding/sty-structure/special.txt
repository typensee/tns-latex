==============================
Organiser des sections de code
==============================

Nous venons de voir que les outils automatisés utilisent des commentaires spéciaux de type "section à la \markdown".
A cela s'ajoute d'autres types de section pour faciliter la maintenance et la compréhension du code \src.
Voici ce qui est disponible.

latex::
    % == TITRE SPÉCIFIQUE POUR L'AUTOMATISATION == %

    % ...

    % -- Titre de niveau 1 pour le codeur -- %

    % ...

    % :: Titre de niveau 2 pour le codeur :: %

    % ...


warning::
    Les titres pour les tâches automatisées sont fixés suivant les cas d'utilisation, donc ils ne peuvent pas être modifiés.
