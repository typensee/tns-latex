==============================
Noms des macros "traductrices"
==============================

Les noms sont toujours construits suivant le même schéma. Voici un exemple éclairant qui vient du package ¨bdoc.
Soit par exemple verb::``exa_title = Example`` que l'on trouve dans path::``src/main/locale/en/sentences.txt``.
Dans path::``EN.sty`` est ajoutée "automatiquement" la macro ``\bdoc@trans@exa@title``.
Le schéma de nommage est du type ``\monpackage@trans@id@une@trad`` où verb::``id_une_trad`` est un identifiant utilisé dans les fichiers path::``PEUF`` de traduction.


info::
    Les outils automatisés gèrent la présence éventuelle de doublons !
