====================
Organiser les \trads
====================

Considérons comme exemple le cas du package \bdoc. Le \bdocdate, son code \src contient un dossier de \dev path::``main`` où l'on trouve les fichiers et dossiers suivants permettant d'avoir des macros gérant le français.

dirtree::
    + main
        + locale
            * french.sty
            + french
                * macros.txt
                * sentences.txt


Voici l'affreux code utilisé en interne pour le français.

latex::
    ---
    path  = example/french.sty
    title = Contenu du fichier path::``french.sty`` le \bdocdate
    ---


Toutes les macros dans l'horreur précédente ont été produites \auto^t
((
    Noter les préfixes utilisés dans le fichier path::``FR.sty`` afin d'éviter des conflits de noms avec d'autres packages (quitte à répéter du code dans différents endroits mais ceci est un moindre mal).
))
à partir des deux fichiers suivants très simples à modifier par un non codeur (la section suivante va préciser la syntaxe utilisée).

verb::
    ---
    path  = example/macros.txt
    title = Contenu du fichier path::``french/macros.txt`` le \bdocdate
    ---

verb::
    ---
    path  = example/sentences.txt
    title = Contenu du fichier path::``french/sentences.txt`` le \bdocdate
    ---


info::
    Autant de fichiers ext::``txt`` que nécessaire pourront être mis à l'intérieur d'un dossier pour une langue comme path::``french`` ci-dessus.


info::
    L'intégration de \nelles langues est expliquée dans la section cf::``?/translate/api-config.txt``.
