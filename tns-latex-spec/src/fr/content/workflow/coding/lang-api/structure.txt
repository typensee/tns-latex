====================
Organiser les \trads
====================

Considérons comme exemple le cas du package \bdoc.
Le \bdocdate, son code source contient un dossier de \dev path::``main`` où l'on trouve les fichiers et dossiers suivants permettant d'avoir des macros gérant le français.

dirtree::
    + main
        + locale
            * FR.sty
            + fr
                * macros.txt
                * sentences.txt


Voici l'affreux code utilisé en interne pour le français.

latex::
    ---
    path  = example/FR.sty
    title = Contenu du fichier path::``FR.sty`` le \bdocdate
    ---


Toutes les macros dans l'horreur précédente ont été produites automatiquement
((
    Noter les préfixes utilisés dans le fichier path::``FR.sty`` afin d'éviter des conflits de noms avec d'autres packages (quitte à répéter du code dans différents endroits mais ceci est un moindre mal).
))
à partir des deux fichiers suivants très simples à modifier pour un non codeur (la section suivante va préciser la syntaxe utilisée).

verb::
    ---
    path  = example/macros.txt
    title = Contenu du fichier path::``fr/macros.txt`` le \bdocdate
    ---

verb::
    ---
    path  = example/sentences.txt
    title = Contenu du fichier path::``fr/sentences.txt`` le \bdocdate
    ---


info::
    À l'intérieur du dossier pourront être mis autant de fichiers path::``TXT`` que nécessaire.


info::
    L'intégration de \nelles langues est expliquée dans la section cf::``?../translate/api-config.txt``.
