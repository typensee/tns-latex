===================================
Un fichier \readme pour les codeurs
===================================

Pour donner une vue globale d'un projet \info, il est de bon usage de fournir un fichier \readme décrivant succinctement le dit projet.
Pour faciliter ceci, \texfacto s'appuie sur \multimd afin de permettre la rédaction du \readme morceau par morceau.
Se reporter à la \doc de \multimd pour en savoir plus.


info::
    Les fichiers \readme sont tous rédigés en anglais.
