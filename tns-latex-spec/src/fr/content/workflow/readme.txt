===================================
Un fichier \readme pour les codeurs
===================================

Pour donner une vue globale d'un projet \info, il est d'usage de fournir un fichier \readme décrivant succinctement le dit projet.
Pour faciliter ceci, \texfacto s'appuie sur \multimd afin de permettre, sans l'imposer, la rédaction du \readme morceau par morceau.
Se reporter à la \doc de \multimd pour en savoir plus.


info::
    Les fichiers \readme sont \tjrs rédigés en anglais.
