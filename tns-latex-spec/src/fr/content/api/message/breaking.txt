===================================
Indiquer des ¨chgts non rétroactifs
===================================

Lors d'un ¨chgt non rétroactif d'¨api, pour ne pas incommoder l'utilisateur, une période fixe de `6` mois permet la vie conjointe des anciennes et ¨nelles interfaces via le mécanisme suivant.

    1) Sans action particulière de l'utilisateur, un message d'erreur, stoppant donc la compilation, indique le ¨chgt d'¨api, mais aussi la possibilité d'user d'une option pour continuer à employer l'ancienne ¨api.

    1) Si le mécanisme d'options pour employer l'ancienne ¨api est activé, un message d'avertissement est utilisé pour rappeler le besoin de changer la syntaxe tout en permettant une compilation du ¨doc.


Une fois la période de `6` mois passée, un mauvais usage sera traité comme une classique erreur sans plus d'¨infos autre que de se reporter à la ¨doc.
