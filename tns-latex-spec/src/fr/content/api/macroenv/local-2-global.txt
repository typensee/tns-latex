=========================================
Des options locales réglables globalement
=========================================

Certaines options de macros, ou d'\envs peuvent être modifiables globalement : on peut par exemple penser au type de décorateurs "ouverts" des intervalles (des parenthèses à l'américaine, ou bien des crochets à la française). Dans ce cas, le réglage est disponible comme option de package, mais aussi via une macro généraliste pour une modification à partir d'un point donné.


info::
    Les réglages globaux de certaines options locales se font via des clés usant de "chemins" faisant \ref aux macros, ou aux \envs concernés. Un exemple fictif pourrait être latex::``\settingtns{nommacro/nomoption = choixfait}``.
