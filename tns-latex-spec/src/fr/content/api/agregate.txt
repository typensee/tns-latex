===========================
Agréger différents packages
===========================

Pour des raisons pratiques, des packages ne font qu'agréger d'autres packages afin d'éviter de devoir importer à la main plein de packages de la suite ¨tnstex.
Ce type de packages agrégateurs proposent tous une ¨api répondant aux spécifications suivantes
((
    La majorité du travail est automatisé.
)).

    * On importe les packages en ne fabriquant pas un package qui reprendrait les codes des différents sous-packages. Ceci simplifie grandement les ¨majs.

    * Les options de chaque sous-package restent accessibles depuis le package agrégateur.

    * Côté utilisateur, il est possible de choisir une importation ciblée de certains packages, ¨etou d'en exclure d'autres.

    * La ¨doc agrège les ¨docs des différents packages importés afin de proposer toutes les ¨infos au même endroit dans un unique ¨pdf.
