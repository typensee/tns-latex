=================
Messages d'erreur
=================

Chaque message d'erreur indique toujours le nom du package qui l'envoie.


info::
    Pour les packages "agrégateurs", les erreurs qui ne leur sont pas propres garderont la référence au sous-package utilisé. Ceci permet de vite cibler un ¨pb, et rend plus efficace la correction d'eventuels bogues.
