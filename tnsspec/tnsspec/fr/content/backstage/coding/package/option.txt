====================
Options des packages
====================

Comme pour les dépendances, les options s'indiquent dans une section spéciale. Voici la structure à suivre.

latex::
    % == OPTIONS == %

    % Lignes codant les options.


info::
    Pour les packages, on s'appuie sur la puissante et sympathique machinerie du package ``l3keys2e``.
